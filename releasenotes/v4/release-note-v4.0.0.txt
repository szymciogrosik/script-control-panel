## App overview
Instructions on how to install and run the application, along with other useful information, can be found [here](https://github.com/szymciogrosik/script-control-panel/wiki).

## Release notes
#### Improvements:
#### Improvements:
* Changed the distribution method. The Script Control Panel App is now distributed as a portable application bundled with the required Java runtime environment.
* Starting with this version, the application is exclusively targeted for Windows.
* Added a layout editor. Layouts and buttons can now be modified directly within the application. This supports both default and custom profiles.
* Updated the additional settings page to support both default and custom profiles.
* Added description labels across all settings pages.
* Added a new NC style.
* File paths in the settings can now be selected using a "Browse" button.
* Upgraded Java from version 21 to 26.
* Upgraded JavaFX from version 11 to 26.

#### Fixes:
* Redesigned the application update mechanism. The app now checks a custom GitHub page for the latest release information. This resolves the GitHub API rate-limiting issue (max 60 requests/hour) previously encountered when multiple users checked for updates from the same office network.

## Warnings
This release is not backward compatible. Configuration files from previous versions must be manually migrated to the new structure detailed [here](https://github.com/szymciogrosik/script-control-panel/wiki/Configuration).
